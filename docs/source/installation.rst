##############
 Installation
##############

The most recent release can be installed from `PyPI
<https://pypi.org/project/pysec2pri>`_ with uv:

.. code-block:: console

    $ uv pip install pysec2pri

or with pip:

.. code-block:: console

    $ python3 -m pip install pysec2pri

*********************
 Installing from git
*********************

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/sec2pri/pysec2pri.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/sec2pri/pysec2pri.git

****************************
 Installing for development
****************************

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/sec2pri/pysec2pri.git
    $ cd pysec2pri
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
